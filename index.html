Weed Classification Project

Project Overview

This project compares various deep learning models such as ResNet50, VGG16, NASNetMobile, MobileNetv2, DenseNet201, EfficientNetB0, InceptionV3, and Xception for weed classification using the Deepweeds dataset from Kaggle. The dataset contains 9 different classes of weeds to be classified. The models were trained on ImageNet weights, with NASNetMobile and DenseNet201 providing notable accuracy.

To bridge the gap between complex machine learning models and everyday users, we developed a web application that allows users to input an image and view the results of weed classification.
