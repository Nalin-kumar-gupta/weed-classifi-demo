Comparative Analysis of Deep Learning Models

Project Overview

This project compares various deep learning models like ResNet50, VGG16, NASNetMobile, and DenseNet201, among others, for the classification of weeds. Using the Deepweeds dataset from Kaggle, which contains 9 different classes of weeds, we trained the models on ImageNet weights. The best-performing models, NASNetMobile and DenseNet201, achieved notable accuracy. To make the technology accessible, we also developed a web app that shows the classification results.
